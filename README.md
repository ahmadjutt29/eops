# EOPS

Welcome to the magnificent EOPS React UI, built with Vite!

This project provides a beautiful and modern front end for EOPS, using React and Vite for fast development and hot module replacement.

## Getting Started

To run the project locally:

```bash
npm install
npm run dev
```

## Features
- Magnificent welcome screen for EOPS
- Modern, responsive, and visually appealing UI
- Built with React and Vite
